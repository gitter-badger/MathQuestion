# MathQuestionHelper
这是一个用于帮助你解决数学问题的程序  
你只需要运行它，并在文本框里编写你的Java代码即可，无需IDE  
## 各区域作用
Import编写区：写import语句
主方法编写区：写你的主要程序，无需声明类和方法，直接编写代码
其他方法编写区：编写你的程序的其他部分，需要方法声明
## 工作原理
本程序会将你写的代码添加进一个默认框架内：  
import代码  
类声明  
&emsp;主方法  
&emsp;&emsp;new Main.go();  
&emsp;&emsp;你编写的主方法  
&emsp;&emsp;&emsp;语句  
&emsp;其他方法  
&emsp;&emsp;&emsp;语句  
结束
